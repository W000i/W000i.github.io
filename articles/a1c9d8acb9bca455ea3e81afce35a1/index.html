T00ls自动签到脚本

2024/8/19 Python

T00ls论坛每天签到和查询域名会给tubi，有时候经常忘记，找了其他人的脚本发现查询域名有问题，简单改了一下。

tips：最近查询域名网站更新，需要改一下获取域名和代码才可以用。

钉钉接入

在群组添加一个自定义的机器人

需要开启自定义关键字，调用接口发送的内容包含关键字才会发出通知

添加成功后复制Webhook

使用下方代码替换webhook_url为你的Webhook，测试机器人是否可用

打码平台

由于采用了cf-turnstile验证方式，需要找支持turnstile的打码平台

经过搜索后发现一家俄罗斯的供应商支持turnstile认证
并且它的价格很实惠，1000次只需要2.99刀

打开官网2captcha
注册账号并充值1美刀，可以使用支付宝或者加密货币进行付款

充值成功后金额会进行刷新
下面的key在调用接口时会使用

安装第三方库
pip3 install 2captcha-python

def t00ls_login(u_name, u_pass, q_num, q_ans):
    """
    t00ls 登录函数
    :param u_name: 用户名
    :param u_pass: 密码的 md5 值 32 位小写
    :param q_num: 安全提问类型
    :param q_ans: 安全提问答案
    :return: 签到要用的 hash 和 登录后的 Cookies
    """
    login_data = {
        'action': 'login',
        'username': u_name,
        'password': u_pass,
        'questionid': q_num,
        'answer': q_ans
    }
    response_login = requests.post('https://www.t00ls.com/login.json', headers=req_headers,data=login_data)
    response_login_json = json.loads(response_login.text)

    if response_login_json['status'] != 'success':
        return None
    else:
        print('用户:', username, '登入成功!')
        formhash = response_login_json['formhash']
        t00ls_cookies = response_login.cookies
        return formhash, t00ls_cookies

# 签到
def t00ls_sign(t00ls_hash, t00ls_cookies):
    """
    t00ls 签到函数
    :param t00ls_hash: 签到要用的 hash
    :param t00ls_cookies: 登录后的 Cookies
    :return: 签到后的 JSON 数据
    """
    sign_data = {
        'formhash': t00ls_hash,
        'signsubmit': "true"
    }
    response_sign = requests.post('https://www.t00ls.com/ajax-sign.json', headers=req_headers,data=sign_data, cookies=t00ls_cookies)
    return json.loads(response_sign.text)

def main():
    response_login = t00ls_login(username, password, question_num, question_answer)
    if response_login:
        response_sign = t00ls_sign(response_login[0], response_login[1])
        if response_sign['status'] == 'success':
            print('签到成功')
            content = '签到成功\n'
            domain_list_log = domains()
            verbose_log = t00ls_domain(response_login[0], response_login[1],domain_list_log[0],domain_list_log[1])
            content += verbose_log
        elif response_sign['message'] == 'alreadysign':
            print('今日已签到')
            content = '今日已签到\n'
            domain_list_log = domains()
            verbose_log = t00ls_domain(response_login[0], response_login[1],domain_list_log[0],domain_list_log[1])
            content += verbose_log
        else:
            print('出现玄学问题了 签到失败')
            content = '出现玄学问题了 签到失败'
    else:
        print('登入失败 请检查输入资料是否正确')
        content = '登入失败 请检查输入资料是否正确'

    """
    钉钉通知函数
    :param content: 要通知的内容
    :return: none
    """
    webhook_url = webhook
    dd_headers = {
        "Content-Type": "application/json",
        "Charset": "UTF-8"
    }
    dd_message = {
        "msgtype": "text",
        "text": {
            "content": f'T00ls 签到通知\n\n{content}'
        }
    }

    requests.post(url=webhook_url, headers=dd_headers, data=json.dumps(dd_message))

if __name__ == '__main__':
    main()
